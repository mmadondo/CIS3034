Hi! Welcome to my small corner on the www! I'm Malvern and I am a junior at the College of St. Scholastica in Duluth, MN.
I study Computer Information Systems and Mathematics.
I am dedicating this site to documenting my progress in CIS 3034 [Client-side Web Development course].
Each week, I will record my progress and include work on my assignments. Stay tuned!

* Click on the [+] to open the collapsible content.

Summary of my experience

Prior to this class, I learned basic HTML and CSS from my jo on campus as Web Assistant for Marketing Team and through various online tutorials. This past summer, I decided to make a full website for a project I am working on. I downloaded a template built with HTML, CSS, and Bootstrap. I went through the code and learned what each line of code was doing, then modifying and improving it to suit my needs.

What I hope to learn in the class

I am looking forward to mastering basic Javascript and a few related frameworks and technologies. I want to be able to make a web extension or web app by the end of the year.

What did you do this past week?

This week I learned how to create an HTML DOM ( document object model) from a web page's source code as well as how to debug a web page using Developer Tools or a text editor such as CodePen. I also learned how web pages are rendered including the roles of the browser, server, HTML, CSS, and Javascript
