Summary of my experience

Prior to this class, I had basic knowledge of HTML5, CSS3, and JavaScript. I learned these from my on-campus job and from working on a side project where I made a full-responsive site using a template with boilerplate code.

What I hope to learn in the class

I am looking forward to learning more about front-end web dev, develop my JS programming skills, Web APIs, different JS frameworks and technologies. I want to be able to make a web extension or web app by the end of the year, even after the class ends.

What did you do this past week?

This week I learned how to create an HTML DOM ( document object model) from a web page's source code as well as how to debug a web page using Developer Tools or a text editor such as CodePen. I also learned how web pages are rendered including the roles of the browser, server, HTML, CSS, and Javascript

What’s in your way?

Finding the best resource(s) to use without overwhelming myself! Having a lot of resources at hand sometimes makes it impossible to even start so I am hoping I will focus on at most 3 main resources that I will refer to as I learn about client-side web dev.

What will you do next week?

Intro to Javascript coming up. I will go through some basic tutorials and a few of the resources shared.

Impressions + Improvements

Debugging is an interesting excercise! It was helpful going through the code on my own and talking to another student about the errors I found and what I did to resolve them. Also learned from listening to how others were debugging. I want to find a way to close all accordions except the one currently open. This is a javascript fix I think so it shouldn't be that hard to figure out //thanks to Google and eh Stack Overflow.

Resources + Tip of the week

Tip: There is a plethora of resources out there and I think its best that I use a few that are more comprehensive and well-rounded than try to learn it all and use each one. Resources I could use include: Javascript.com, Eloquent Javascript, Mozilla Developer's Network, The New Boston and JSON website.

What did you do this past week?
